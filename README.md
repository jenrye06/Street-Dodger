# Crossy Road–Inspired Game

A browser-based arcade game inspired by Crossy Road where players navigate through moving traffic, avoid collisions, and survive for as long as possible.

## Technologies

- Python
- Brython
- HTML
- CSS

## Project Overview

This project was created during my junior year of high school after learning Python. Wanting to apply what I had learned in a larger project, I designed and developed a game inspired by *Crossy Road*. Players move across multiple lanes of traffic while avoiding vehicles with randomized speeds, colors, and directions.

While originally developed in the CodeHS environment, this repository has been refactored and adapted to run outside of CodeHS, requiring significant changes to the original implementation.

## Features

- Randomized vehicle colors, speeds, and travel directions
- Multiple traffic lanes with independent behavior
- Collision detection and game over logic
- Survival timer that tracks how long the player remains alive
- Custom explosion animation upon collision
- Keyboard-controlled player movement
- Spawn logic that prevents vehicles from overlapping

## What I Learned

- Structuring larger Python programs using functions
- Managing game state and event-driven programming
- Implementing collision detection algorithms
- Using randomized game mechanics to improve replayability
- Debugging and refactoring code when migrating a project between development environments

## Repository Notes

This repository is a refactored version of the original CodeHS project. The game was migrated to a standard GitHub project structure and modified to replace CodeHS-specific functionality while preserving the original gameplay.
